# b4. Triangoli
```python
# Leggo i numeri e li trasformo in intero
numero = float(input("Inserisci il primo numero:"))

# Se il resto della divisione per 2 è zero il numero è pari
if numero % 2 == 0:
    print("Pari")
else: #altrimenti...
    print("Dispari")
```
