# b1. Facciamo ordine
```python
# Leggo i numeri e li trasformo in float
primo = float(input("Inserisci il primo numero:"))
secondo = float(input("Inserisci il secondo numero:"))

# Se il primo è il più piccolo ...
if primo < secondo:
    print(primo)
    print(secondo)
else: #altrimenti...
    print(secondo)
    print(primo)
```
